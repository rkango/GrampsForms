# Gramps Forms

This repository contains various forms that I have created for the Gramps Forms plugin.

## Current Forms Covered
* IE1926 (1926 Irish Free State Census, being released on 2026-04-18)
* IEBirth (Irish Birth civil records)
* IEDeath (Irish Death civil records)
* IEMarriage (Irish Marriage civil records)
* NYCBirth (New York City birth certificate, Historical Vital Records)
* NYCDeath (New York City death certificate, Historical Vital Records)
* NYCMarCert (New York City marriage certificate, Historical Vital Records, Department of Health)
* NYCMarLic (New York City marriage license/certificate, Historical Vital Records, City Clerk)
* SC1911 (1911 Scottish Census)
* SC1921 (1921 Scottish Census)
* SCBirth (Scottish Birth register)
* SCDeath (Scottish Death register)
* SCMarriage (Scottish Marriage register)
* USDclInt (US Declaration of Intention, also known as "first papers")
* USPetNat (US Petition for Naturalization)
* USPasMnf (US Passenger Manifest)

## How To Install
1. Download the test.xml file in this repository.
2. Go to the Form plugin directory (for Gramps 6.0 on Windows, it is under ```%AppData%\gramps\gramps60\plugins\Form```).
3. Place the test.xml file in the directory.
4. Reboot Gramps.
5. Go to your source and add the attribute Form with the appropriate value above.
